# Vincent Mizhquiri - Portfolio

A professional portfolio website showcasing my experience as a Talent Acquisition Specialist with expertise in fintech, ecommerce, and AI-driven recruitment strategies.

## Overview

This portfolio website highlights my 10+ years of experience in talent acquisition, including work with Fortune 500 companies and innovative startups across financial services, fintech, healthcare, ecommerce, and government sectors.

## Features

- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Smooth Navigation**: Interactive navigation with smooth scrolling
- **Contact Form**: Functional contact form for inquiries
- **Professional Sections**: 
  - About Me
  - Work Experience
  - Contact Information
  - Social Media Links

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome Icons

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/vincentmizhquiri/VincentMIzhquiri-Portfolio.git
   ```

2. Open `index.html` in your web browser

3. For local development, you can use a local server:
   ```bash
   # Using Python
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```
